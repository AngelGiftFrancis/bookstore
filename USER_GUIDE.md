## 1. Introduction
The Bookstore app allows users to browse, buy, and manage books. Admin users can also manage the inventory.

## 2. Prerequisites / System Requirements
- Flutter SDK
- Dart SDK
- Android Studio or VS Code
- Internet connection (for online backend)

## 3. Getting Started (Installation & Launch)
1. Clone the repo: git clone https://github.com/AngelGiftFrancis/bookstore.git
2. Navigate to the project folder: cd bookstore
3. Install dependencies: flutter pub get
4. Run the app: flutter run

## 4. Features & How to Use Them
- User Authentication (Sign Up/Login/Logout/Password Reset)
- Browse books
- Search and filter
- Shopping cart & checkout
- Wishlist
- User Profile Management
- Order Management
- Admin Panel (if admin)
- Download invoices
